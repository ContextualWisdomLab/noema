# Noema 이용약관(초안)

## 서비스 범위
- Noema는 GitHub Actions OIDC 토큰을 검증하고 GitHub App 설치 토큰으로 교환하는 API 서비스입니다.
- 토큰은 Noema 정책 내에서만 사용되며, 고객 저장소의 권한 범위는 승인된 App 권한 한정입니다.

## 이용 조건
- 고객은 `ALLOWED_*` 정책값을 유지하고 중앙 워크플로 규칙을 변경할 경우 사전 통지해야 합니다.
- 고객이 OIDC 서명 키 또는 App 비밀키를 유출한 경우 즉시 통지합니다.

## 면책·책임
- GitHub 서비스 장애, 네트워크 장애 등 외부 요인으로 인한 지연은 가용성 대상에서 별도 산정합니다.
- Noema는 GitHub API 정책 변경으로 인한 장기 장애에 대해 사전 공지를 우선합니다.

## 종료 및 회수
- 보안 사고 시 Noema는 접근을 일시 중단하고 토큰/비밀키 회수 지원을 제공합니다.
- 계약 종료 시 관련 비밀키 폐기 절차를 수립해야 합니다.

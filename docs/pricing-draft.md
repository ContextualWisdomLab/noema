# Noema 가격 모델(초안)

## 무료/파일럿
- 파일럿: 1개 조직, 월 5천 요청 / 월, 지원 응답 SLA 2일
- PoC/파일럿 연장: 사전 승인 시 기간 연장

## 유료
- Starter: 10만 요청/월, 월 정액 기준
- Team: 50만 요청/월 + SSO 연계 옵션
- Enterprise: 커스텀 한도 및 전담 운영

요금 산정은 토큰 교환 요청 수를 기준으로 시작하고, 조직별 계약량에 따라 선불 예치 또는 사용량 과금 중 하나를 선택합니다.

## 추가 항목
- 추가 요청 패키지: 1만 요청 단위
- 긴급 지원(24h 대응) 플랜 추가료 별도
